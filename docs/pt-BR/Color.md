
# Color

A classe [Color](../../source/common/Color.mjs) é um auxilio para operações com os valores em bytes das cores.

> [!WARNING]
> Essa classe é uma extenção de [Uint8Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Uint8Array).


Os sistemas operacionais possuem estruturas diferentes de armazenar os bytes na memoria, conhecido como [Endianness](https://en.wikipedia.org/wiki/Endianness).
